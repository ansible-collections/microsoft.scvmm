microsoft.scvmm Release Notes

.. contents:: Topics

v1.0.0
======

Release Summary
---------------

Initial release of the microsoft.scvmm collection for System Center Virtual Machine Manager (SCVMM) 2022 automation.

New Modules
-----------

- scvmm_vm - Create and manage virtual machines
- scvmm_vm_info - Gather information about virtual machines
- scvmm_vm_state - Manage VM power state
- scvmm_vm_checkpoint - Manage VM checkpoints
- scvmm_vm_migrate - Migrate VMs between hosts
- scvmm_vm_clone - Clone virtual machines
- scvmm_vm_dvd_drive - Manage VM DVD drives
- scvmm_vm_scsi_adapter - Manage VM SCSI adapters
